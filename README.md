# Light & Soul - Wellness Website

Een Next.js website voor Light & Soul, een praktijk voor healing, emotioneel lichaamswerk en astrologie.

## Over het project

Deze website is gebouwd met Next.js 16, TypeScript en Tailwind CSS. Het bevat informatie over de diensten van Beata van Etten, inclusief contactgegevens, privacyverklaring en disclaimer.

## Ontwikkeling

### Vereisten
- Node.js 18+
- npm

### Installatie
```bash
npm install
```

### Ontwikkelingsserver starten
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in je browser.

### Build
```bash
npm run build
```

### Start productie versie
```bash
npm start
```

## Project structuur

- `app/` - Next.js App Router pagina's
- `components/` - Herbruikbare React componenten
- `public/` - Statische assets

## Styling

De website gebruikt een warme, persoonlijke uitstraling met:
- Sage groen (#9CAF88)
- Warm ivoor achtergrond (#f5f5dc)
- Zachte goud accenten (#D4AF37)
- Lettertypes: Cormorant Garamond (serif) en Mulish (sans-serif)

## Deployment

De eenvoudigste manier om te deployen is via Vercel of een andere hosting provider die Next.js ondersteunt.
